                  # 3.1 通用设置

                  基本都在 setting -> editor 中找

                  背景图： 双击 Shift -> set background

                  • lint： 用于检查代码规范（需下载安装&配置） Pycharm 也会自动检查，波浪线表示有问题

                  • 快捷注释：ctrl + /

                  • 查看保留字

                    from keyword import kwlist

print([i for i in kwlist])

### 输出结果：
# ['False', 'None', 'True', '__peg_parser__', 'and', 'as', 'assert', 'async', 'await',
# 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally',
# 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not',
# 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']

                    注： 区分大小写 不能以保留字命名

                  注意其他应用 热键占用，如 网易云

                  • 批量的快速重命名：Shift + F6
                  • 快速查找文件名：Shift + Ctrl + N

                  快捷键大全： Help - Keymap Reference
